# Brand Lock (Do Not Drift)

Use this as a guardrail during homepage rebuild.

## Non-negotiables
- Keep existing visual identity and design language.
- No new aesthetic system without explicit approval.
- Keep existing font families and hierarchy style unless brief explicitly overrides copy/layout.
- Keep current color system and button styling conventions.
- Preserve overall AF tone: dark, technical, institutional, security-first.

## Allowed changes
- Section structure and content per approved brief.
- CTA labels and destinations where requested.
- New modules that match current visual system.

## Process discipline
- One section at a time.
- Commit + push after each section.
- Vercel preview check before moving to next section.
